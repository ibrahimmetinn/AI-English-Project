# GramLab — Assets Klasörü

Bu klasöre medya dosyalarını ekle:

## Müzik
- `mafia-music.mp3` → The Family bölümü arka plan müziği

## Gelecekte eklenebilecekler
- `gate1-sound.mp3` → 1. Kapı geçiş sesi
- `gate2-sound.mp3` → 2. Kapı geçiş sesi
- `achievement.mp3` → Rozet kazanma sesi
- `correct.mp3`     → Doğru cevap sesi
- `wrong.mp3`       → Yanlış cevap sesi

## Kullanım
Müzik dosyasını bu klasöre koy, adı `mafia-music.mp3` olsun.
Farklı isim kullanmak istersen index.html'de:
  _mafiaSetMusicSrc('yeni-isim.mp3')
